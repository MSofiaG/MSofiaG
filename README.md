## 🚀 Featured Projects

### ⚙️ AutoFlow — Visual Workflow Automation Platform
A visual process automation platform (inspired by tools like Zapier/n8n) where users build automation pipelines through an interactive drag-and-drop canvas.

- 🎨 Interactive drag-and-drop flow canvas
- 🔄 Custom BFS-based workflow execution engine
- 📨 Asynchronous execution via RabbitMQ
- 📊 Execution logs & flow history
- 📄 Full CRUD for flows and executions, documented with OpenAPI

`React` `Spring Boot` `PostgreSQL` `RabbitMQ` `Docker`

[🔗 Repository](https://github.com/MSofiaG/autoflow) · [🖥️ Demo](#)

### 💪 GymTrack AI — AI-Powered Fitness Tracker
A fitness tracking platform that helps users monitor workouts, visualize progress, and generate personalized training plans using AI.

- 🔐 JWT authentication & onboarding based on fitness goals
- 🏋️ Catalog of 60+ exercises with goal-based workout templates
- 📈 Weekly/monthly dashboards, progress charts & personal records
- 🤖 AI-assisted workout generation and plateau detection *(in progress)*

`Java 17` `Spring Boot` `Spring Security` `PostgreSQL` `JWT` `React` `Vite` `Tailwind CSS` `Recharts`

[🔗 Repository](https://github.com/MSofiaG/gymtrack-ai) · [🖥️ Demo](#)
